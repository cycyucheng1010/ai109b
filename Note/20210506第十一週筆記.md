## 常用函式庫
* matplotlib -----> 畫圖、視覺化工具
* numpy -------> 數學矩陣運算
  * import numpy as np
  * ```np.arange(x,y)``` 從起始值到上限值
  * ```np.add(x,y)``` 兩矩陣相加
  * ```.shape()``` 回傳矩陣為幾乘幾陣列  ```.shape=(x,y)``` 重設矩陣維度
  * ```np.linspace(x,y,z)```  從x~y分成z個
  * 
* scipy -----> 擴充numpy
* sympy -----> 符號運算
